# Fugue Icons

See [the icons GitHub page][icons-gh] and [the icons homepage][icons-home].


## Install

Download [manually](https://github.com/johnnyt/fugue-icons/releases) or with bower.

#### [Bower](http://bower.io)

```
bower install --save fugue-icons
```
